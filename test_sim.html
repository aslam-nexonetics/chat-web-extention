Auth & Profile Simulation

This page simulates the full authentication lifecycle of the Nexonetics extension.

How to test:

  • 1. Click the floating chat icon (bottom-right).
  • 2. Enter any credentials and click Login.
  • 3. Watch it transition to the Chat view with a mocked user profile.
  • 4. Notice the Avatar and Full Name in the header.
  • 5. Click the Logout icon to return to the login screen.
